## Python

### 1. 코드 작성
* 프로그래머스에서는 다음과 같은 형식으로 코드를 작성하도록 나와있지만,
> **answer =** a+b
> 
> **return answer**

아래 형식으로도 작성 가능
> **return** a+b

### 2. 제한 조건
* 코딩 테스트나 연습 문제에서는 제한 조건을 코드에 따로 구현할 필요는 없지만, 실무 코드거나 협업 중일 때는 제한 조건을 주석으로 명시하거나 검증하는 코드 필요할 수 있음
